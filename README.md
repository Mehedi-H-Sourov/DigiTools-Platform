# 🚀 Digi-Tools - Your Ai Hub

A modern web platform designed to simplify how users explore and manage premium AI tool subscriptions.  
This project combines a **professional design made with React** with a **smooth and intuitive user experience**.

---

## 📌 About the Project

**Digi-Tools** is a high-performance web application that showcases a curated marketplace of AI-powered tools and services.  
The goal was to build a platform that feels fast, clean, and responsive — allowing users to browse, compare, and select digital products effortlessly across all devices.

---

## 🛠 Tech Stack

- **React.js** → Component-based architecture for scalable UI  
- **Tailwind CSS + DaisyUI** → Utility-first styling with pre-built UI components  
- **JavaScript (ES6+)** → Modern syntax (arrow functions, destructuring, async/await)  
- **HTML5** → Semantic and accessible structure  
- **Vite** → Lightning-fast development and build tool
